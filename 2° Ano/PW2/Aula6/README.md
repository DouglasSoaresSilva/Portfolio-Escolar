# Aula 6 - Projeto PHP

Projeto desenvolvido na Aula 6 da disciplina de Programação Web 2 (PW2).

## 📋 Descrição

Este é um projeto educacional voltado para o aprendizado de desenvolvimento web com PHP. O projeto utiliza PHP como linguagem principal, com CSS para estilização e HTML para a estrutura.

## 🛠️ Tecnologias Utilizadas

- **PHP** - Linguagem principal
- **CSS** - Estilização e layout
- **HTML** - Estrutura das páginas

## 📁 Estrutura do Projeto

```
Aula6/
├── README.md
├── Aula6.md
└── [outros arquivos do projeto]
```

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/DouglasSoaresSilva/PW2.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd PW2/Aula6
   ```

3. Execute o projeto em um servidor PHP local:
   ```bash
   php -S localhost:8000
   ```

4. Acesse no navegador:
   ```
   http://localhost:8000
   ```
