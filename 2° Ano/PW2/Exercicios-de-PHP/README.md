## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/DouglasSoaresSilva/Exercicios-de-PHP.git
```

2. Navegue até a pasta do projeto:
```bash
cd Exercicios-de-PHP/EXERCICIOS
```

3. Inicie um servidor PHP local:
```bash
php -S localhost:8000
```

4. Abra seu navegador e acesse:
```
http://localhost:8000
```
