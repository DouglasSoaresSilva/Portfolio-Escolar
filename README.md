# 📚 Portfólio Escolar
 
**DS-AMS · Etec Zona Leste / Escola Técnica**
 
Uma coleção organizada de todas as tarefas, projetos e exercícios desenvolvidos ao longo do curso de Desenvolvimento de Sistemas.
 
---
 
## 📖 Sobre o Portfólio
 
Este repositório possui todos os materiais produzidos durante o curso **DS/AMS** (Desenvolvimento de Sistemas/Articulação Médio e Superior), organizado por semestre e disciplina. O objetivo é registrar a evolução do aprendizado e manter um histórico de tudo que foi desenvolvido ao longo do ano letivo.
